中国近代史纲要

中华民国

正文部分

一、中华民国临时政府

  1. 建立：

    1912年1月，中华民国临时政府在南京成立

  2. 《中华民国临时约法》的地位：

    中国历史上第一部具有资产阶级共和国宪法性质的重要文件

二、北洋军阀统治时期

  1. 特征：

    军阀混战，政局动荡

  2. 袁世凯复辟帝制

    • 过程：

      • 从实行责任内阁制到总统制

      对日本出卖国家利益，签订《二十一条》，为日本支持其复辟帝制争取条件

      复辟帝制，尊孔复古

    • 结果：

      护国战争的开展，迫使袁世凯取消帝制

  3. 民族资本主义的发展

    • 原因：

      • 民国建立，扫除了政治上的一些束缚和障碍

      南京临时政府成立后，鼓励民间兴办实业

      一战时期，西方列强忙于战争无暇东顾

      群众性的反帝爱国斗争的有力推动

    • 表现：

      • 纺织、面粉等轻工业发展迅速；一批民族企业迅速壮大

      中国产业工人人数急剧增加，成为不可忽视的社会力量

  4. 新文化运动的开展

    • 兴起原因：

      人们认识到辛亥革命失败在于大多数国民缺乏对旧思想、旧文化、旧礼教的彻底批判，仍被专制和愚昧束缚

    • 开始标志：

      1915年9月陈独秀创办《青年杂志》

    • 主要主张

      • 前期：民主与科学

      后期：宣传马克思主义

    • 历史影响：

      高举民主与科学的旗帜，推动思想文化的革新，有着解放思想的重大意义

  5. 五四运动

    • 背景：

      • 国内

      政治上：北洋军阀统治黑暗，社会阶级矛盾尖锐

      经济上：民族资本主义进一步发展，工人阶级壮大

      思想上：新文化运动和马克思主义的传播

      国外

      美、日加紧侵略中国，反帝爱国运动斗争此起彼伏

      俄国十月革命的影响

      导火索：巴黎和会上中国外交失败

    • 结果：

      释放被捕学生，拒绝在和约上签字

    • 性质：

      是一场彻底的不妥协的反帝反封建爱国运动

    • 特点：

      • 革命性：反帝反封建的革命运动

      爱国性：维护主权，反对帝国主义的侵略

      群众性：学生、工人、商人多阶层参加，联合斗争

      彻底性：斗争目标基本实现

    • 评价：

      • 使青年知识分子得到洗礼

      推动马克思主义广泛传播

      为中国共产党成立做了思想上和干部上的准备

      标志新民主主义革命的开端

  6. 中国共产党诞生

    • 条件

      • 阶级条件：工人阶级的发展壮大

      思想条件：马克思主义的广泛传播

      组织条件：各地共产党早期组织的建立

    • 召开：

      在上海召开，会议最后一天转移到浙江嘉兴南湖的游船上

    • 意义：

      中国共产党诞生，从此中国革命的面貌焕然一新

  7. 第一次国共合作与国民革命

    • 国共合作

      • 原因：中国共产党在开展工人运动的同时，认识到必须建立革命统一战线

      实现：1924年，孙中山在国民党“一大”提出新三民主义，通过了联俄、联共、扶助农工三大政策，第一次国共合作正式形成

    • 国民革命运动

      • 目标：打倒列强，除军阀

      原因：陈独秀等人右倾（主观原因）；蒋介石，汪精卫发动反革命政变（客观原因）

三、南京国民政府

  1. 政治上

    • 国民政府推行独裁专制统治

    • 日本发动侵华战争

    • 国共联合抗日

  2. 经济上：民族资本主义取得较快发展

    3. 原因：

    • 国民经济建设运动的开展

    • 缓和了世界经济危机带来的影响

    • 使资本主义取得较快发展

    • 为持久抗战奠定了基础

    • （南京国民经济与甲午战后经济的共同点：政府政策的支持）

    • 实业救国的思潮

    • 反帝爱国运动

  3. 工农武装割据革命开辟革命新道路

    • 武装反抗

      • 南昌起义：1927年8月1日，打响了反抗国民党反动派第一枪

      八七会议：1927年8月7日确定实行土地革命和武装反抗国民党反动派的总方针，提出“政权是由枪杆子中取得”的重要思想

    • 开辟革命新道路

      • 井冈山革命根据地：（1）1927年创立第一个农村革命根据地

  4. 长征：

    • 背景：

      共产国际的指导脱离中国革命实际，中共中央犯了“左”倾错误，导致第五次反“围剿”失利

    • 转折：

      1935年1月遵义会议召开，开始确立以毛泽东为主要代表的马克思主义正确路线在党中央的领导地位，挽救了中国革命

    • 意义：

      实现了红军的战略大转移，宣传了中国共产党的政治主张

  5. 中华民族的抗日战争

    • 局部抗战

      • 根本原因：日本资本主义发展的需要

      历史原因：“征服满蒙”是日本的既定国策

      客观原因：西方大国实施绥靖政策

      直接原因：受资本主义经济危机的影响

      现实原因：日本国内资源匮乏

      思想原因：日本军国主义思想浓厚

      有利时机：国民政府全力“围剿”红军，给日本侵华以可乘之机；西方主要资本主义国家陷于经济大危机中，无暇他顾

      积极准备：1931年上半年，日本加紧进行武装侵略的部署

    • 中国各界的回应

      • 国民政府：推行“攘外必先安内”方针

      中国共产党：

      1935年发表“八一宣言”，号召“停止内战，一致抗日”

      1935年年底，瓦窑堡会议确定了建立抗日民族统一战线的方针，统一了党内思想

      华北事变：1935年，策动所谓“华北自治运动”等一连串事件，又阶级矛盾上升到民族矛盾

      西安事变：

      背景：在中国共产党抗日民族统一战线政策的感召下，张学良和杨虎城在西北地区实行联共抗日

      意义：西安事变的和平解决，成为扭转时局的枢纽，十年内战的局面基本结束，全国团结抗战的局面初步形成

    • 全面抗战

      • 七七事变：1937年7月7日，日军进攻宛平城与卢沟桥制造了七七事变，成为全面抗战的开端

      抗日民族统一战线：1937年9月22日，国民党发表了中共提出的国共合作抗战宣言，承认中共合法地位，抗日民族统一战线正式形成

    • 抗日战争的胜利

      • 中共七大

      内容：提出建立一个新民主主义的中国；确立毛泽东思想为党的指导思想

      意义：使全党在马克思列宁主义、毛泽东思想的基础上达到了空前的团结

      日本投降：1945年8月15日，日本无条件投降

      意义：是近代以来中国抗击外敌入侵所取得的第一次完全胜利
      重新确立了中国在世界上的大国地位
      大大增强了全国人民的民族自尊心和自信心
      开辟了中华民族伟大复兴的光明前景

  6. 解放战争

    • 重庆谈判：

      1945年10月，国共双方签署《双十协定》，内容为：“避免内战、建设独立、富强和自由的新中国”

    • 全面内战的爆发：

      • 爆发：1946年6月，国民党围攻中原解放区

      粉碎全面进攻：1946年7月~10月，人民解放军粉碎国民党的全面进攻

    • 新民主主义革命的胜利

      • 战略决战：1948年9月~1949年1月进行了三大战役，国民党主力基本上被摧毁

      渡江战役：1949年4月，解放军横渡长江，占领南京，国民党蒋介石集团在大陆的统治覆灭

    • 中共七届二中全会

      • 时间：1949年春，河北西柏坡

      内容：指出党的工作重心必须由乡村转向城市

    • 新民主主义革命的胜利的意义

      • 是马克思主义普遍原理与中国革命具体实践相结合的胜利，是毛泽东思想的胜利

      从根本上改变了中国社会的发展方向，是20世纪人类历史上最具影响的伟大事件之一

主要内容

一、认识共产党成立的意义

  1. 新的领导核心：

    革命性最强的共产党是无产阶级的先锋队。中国共产党的诞生，使中国革命有了坚强的领导核心

  2. 新的指导思想：

    中国共产党是一个以马克思主义为基础的先进政党，能够制定出彻底的反帝反封建的民主革命纲领和正确的策略方针

  3. 新的群众基础：

    • 中国共产党不仅代表广大工人阶级的利益，还代表广大中国人民的利益

    • 由中国共产党领导中国革命，能得到中国各族人民的支持

  4. 新的国际环境：

    中国共产党把中国革命和无产阶级革命联系，是中国革命获得广泛的国际援助，特别是得到了国际共产的帮助

  5. 新的革命成就：

    • 中国共产党的成立，给灾难深重的中国人民了光明和希望，给中国革命指明了方向

    • 实现了国共合作，建立了革命统一战线，推动并参与领导了国民革命运动，中国革命出现了前所未有的新面貌

二、国民革命运动的特点及影响

  1. 特点

    • 革命任务：

      反帝反封建

    • 群众基础：

      四个阶级的联合统一战线

    • 指导思想：

      马克思主义和新三民主义两大革命指导思想并存

    • 革命军队：

      组建国民革命军，进行北伐战争，基本消灭了封建军阀，打击了列强的侵华势力

  2. 影响：

    • 组建国民革命军，进行北伐战争，基本消灭了封建军阀，打击了列强的侵华势力

    • 宣传了中国共产党反帝反封建的革命纲领，扩大了中国共产党在群众中的影响

    • 共产党开始掌握了一部分革命武装

    • 广大群众受到一次革命的洗礼，为中国革命继续前行奠定了基础

三、抗日民族统一战线的特点

  1. 广泛的民族性

  2. 极大的复杂性

  3. 战略目标的集中性

  4. 组织形式的不稳定性

  5. 存在时间的长期性

  6. 受国际环境的影响

四、抗日战争的基本特点

  1. 旗帜鲜明：

    建立了以国共合作为基础的抗日民族统一战线

  2. 持久性：

    抗日战争持续长达14年

  3. 民族性：

    全国各阶级、阶层，工农兵学商各界、各族人民，社会各阶层爱国人士和海外侨胞，在抗日的基础上团结一致，集中力量争取民族解放战争的胜利

  4. 两个战场：

    正面战场和敌后战场

  5. 中国共产党起到中流砥柱的作用：

    积极促成了抗日民族统一战线的建立；坚持全面抗战路线和持久抗战方针；坚决维护抗日民族统一战线

  6. 以弱胜强：

    敌强我弱，抗日战争是近代以来中国抗击外敌入侵取得的第一次完全胜利

  7. 双重性：

    从性质上看，既是反法西斯战争，又是民族解放战争，具有双重性

五、人民解放战争迅速取得胜利的原因

  1. 原因：

  2. 从国民党方面说：

    政治孤立、失去民心；经济崩溃、物价暴涨；军队厌战、士气低落；贪官横行、统治腐败

    • 从共产党角度说

    • 灵魂：

      坚持毛泽东思想的正确指导

    • 核心：

      中国共产党的正确领导

    • 道路：

      走经新民主主义向社会主义发展的道路

    • 基础：

      广大人民群众支持

    • 中坚力量：

      有一只人民的革命军队，坚持武装斗争

    • 保障：

      组成了广泛的统一战线，团结了一切可以团结的力量，共同奋斗

六、新民主主义革命时期中国共产党领导革命探索的特点

  1. 从最高纲领到最低纲领：

    国共第一次合作，开展国民革命，打倒帝国主义，推翻军阀统治，统一中国

  2. 从以城市为中心到创建农村革命根据地：

    国共十年对峙时期，工农武装割据，开辟农村包围城市的革命道路

  3. 从反蒋抗日到联蒋抗日：

    国共第二次合作，开展全民族抗日战争，取得抗日战争的胜利

  4. 从农村再到城市：

    人民解放战争，从战略防御、战略进攻，到三大战役、战略决战、渡江战役，推翻国民党在大陆的统治，工作中心从农村转到城市
